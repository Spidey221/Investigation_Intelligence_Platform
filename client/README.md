# IIP Client

This directory contains the React and Vite frontend for the Investigation Intelligence Platform.

## Architecture
- **Components**: Feature-based organization (`cases/`, `evidence/`, `layout/`) with barrel exports.
- **Pages**: Top-level route components.
- **API**: Axios instance and endpoint functions.

## Setup
1. Run `npm install`
2. Run `npm run dev` to start the Vite development server.
